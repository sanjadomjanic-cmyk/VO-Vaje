# 🛡️ CTF Labs – Cybersecurity Exercises

Welcome to the **Capture The Flag (CTF) lab series**, designed for students of cybersecurity.  
These labs provide hands-on experience with real-world security concepts through practical challenges.

## 📚 Overview

This repository contains two CTF exercises:

- `ctf01/` – Introductory challenges focusing on basic security concepts  
- `ctf02/` – More advanced challenges requiring deeper analysis and problem-solving  

Each lab is structured as a set of tasks where you must discover hidden information (flags) by applying cybersecurity techniques.

---

## 🎯 Learning Objectives

By completing these labs, you will:

- Understand common security vulnerabilities  
- Practice basic penetration testing techniques  
- Learn how to analyze systems and identify weaknesses  
- Improve problem-solving and investigative skills  
- Gain hands-on experience in a safe, controlled environment  

---

## 🧩 What is a CTF?

A **Capture The Flag (CTF)** is a cybersecurity competition where participants solve challenges to find hidden "flags" (usually strings of text).

Flags typically look like: FLAG{example_flag}

---

## ⚙️ Requirements

- Basic knowledge of:
  - Linux command line
  - Networking fundamentals
  - Web technologies (HTTP, browsers)
- Recommended tools:
  - Kali Linux or Ubuntu
  - Browser developer tools
  - `nmap`, `curl`, `grep`, `strings`, etc.
